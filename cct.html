Cooling Tower — Reciprocal Function

y = 1/x

Parent Function

y = 1/x

Transformed Function

y = 1/(x − 0.6) − 0.4  {0.8 < x < 3}

Transformations

To model the right edge of the cooling tower, I started with the basic reciprocal function y = 1/x and made two adjustments. I shifted it 0.6 units to the right by replacing x with (x − 0.6), which moved the vertical asymptote to sit at x = 0.6 — right along the centre of the tower. I also dropped the whole curve down by 0.4 units, pushing the horizontal asymptote to y = −0.4 so it lines up better with the base of the tower. The way the cooling tower narrows in the middle and flares outward is a natural fit for this function, since the reciprocal curve drops steeply near the asymptote and levels out further away. I restricted the domain to between 0.8 and 3 so the curve only shows the visible part of the tower's outline.

Domain & Range

The domain is {x ∈ ℝ | 0.8 < x < 3}. In real-world terms, this covers the horizontal distance from roughly 0.8 metres to 3 metres out from the centre of the tower — exactly the portion of the right edge visible in the photo. The range is {y ∈ ℝ | −0.4 < y < 4}, representing the height of the tower's outline above ground across that section. The lower bound of −0.4 is where the horizontal asymptote sits after the vertical shift, and the upper bound of 4 is approximately where the visible edge of the tower reaches at its tallest point in the image.

— Reciprocal Squared Function

y = 1/x²

Parent Function

y = 1/x²

Transformed Function

y = 2.2/(x − 0.3)² − 1.5  {−2 ≤ x ≤ 2}

Transformations

Starting from the parent function y = 1/x², three transformations were applied to fit the shape in the photo. First, the graph was shifted 0.3 units to the right by replacing x with (x − 0.3), which moved the vertical asymptote from x = 0 to x = 0.3, aligning the curve better with the subject. Second, a vertical stretch of 2.2 was applied, making the two branches of the curve taller and narrower so they matched the shape more closely. Third, the whole graph was brought down 1.5 units, shifting the horizontal asymptote to y = −1.5. The result is a symmetric curve with two upward-opening branches sitting on either side of the asymptote at x = 0.3.

Domain & Range

The domain is {x ∈ ℝ | x ≠ 0.3, −2 ≤ x ≤ 2}, covering the full horizontal width of the subject in the photo while excluding the asymptote at x = 0.3 where the function is undefined. The range is {y ∈ ℝ | −1 ≤ y ≤ 5}, which captures the vertical spread of both branches within the restricted domain — the lower bound of −1 comes from the downward shift, and the upper bound of 5 is where the curve peaks closest to the asymptote.

Function 03

MH

Rollercoaster — Polynomial Function

Degree > 2

Parent Function

y = x³

Transformed Function

y = 0.3(x+1.8)(x−2.3)(x−0.1) + 3  {−2.5 < x < 3}

Transformations

A rollercoaster track is a great real-world example of a higher degree polynomial because of the way it rises, peaks, and then comes back down — exactly what a cubic function does. I wrote the equation in factored form to control where the curve crosses zero. The three roots were placed at x = −1.8, x = 0.1, and x = 2.3, setting the curve's turning points to match the shape of the track in the photo. A vertical compression of 0.3 was applied so the curve wouldn't be too steep, and I shifted the whole thing up 3 units so the track stays above the x-axis — since the rollercoaster is always above ground. The negative leading coefficient flips the curve so it rises then falls, matching the visible drop of the track. The domain is restricted to between −2.5 and 3 to only show the section of track in the photo.

Domain & Range

The domain is {x ∈ ℝ | −2.5 < x < 3}, representing roughly 5.5 metres of horizontal track length from start to end of the visible section. The range is {y ∈ ℝ | 0 < y < 4.5}, covering the height of the track from just above ground level up to its peak at around 4.5 metres. The lower bound being above zero makes sense physically since the track never touches the ground in the photo, and 4.5 metres is approximately the highest point the track reaches at its crest.

Function 04

Wil

— Trigonometric Function with Asymptote

y = cot(x)

Parent Function

y = a·cot(b(x − h)) + k

Transformed Function

y = 30·cot(0.2(x + 5.2))  {−9 ≤ x ≤ 9}

Transformations

The parent cotangent function was transformed in three ways to fit the shape in the photo. A vertical stretch of 30 was applied, dramatically increasing the steepness of the curve near the asymptotes so it shoots up and drops down much more sharply than the basic cot(x) would. A horizontal stretch was introduced through the 0.2 coefficient inside the function, which widens the period from π to roughly 15.71 units — spreading the curve out across a much larger x range and giving each branch a long, gentle descent. The graph was also shifted 5.2 units to the left, repositioning the asymptotes from their default positions at multiples of π to x = −5.2 + 5πn, aligning them with the subject in the photo. No vertical shift was applied, so the midline stays at y = 0 and the curve crosses the x-axis exactly halfway between each pair of asymptotes.

Domain & Range

The domain is restricted to {x ∈ ℝ | −9 ≤ x ≤ 9}, which captures the horizontal span of the subject visible in the photo and falls between two asymptotes so the function is continuous throughout. The range is {y ∈ ℝ | −2 ≤ y ≤ 1}, capturing the vertical spread of the curve within the visible region — starting high near one asymptote and decreasing sharply toward the other end of the domain.

Function 05

Blake

— Sinusoidal Function

y = cos(x)

Parent Function

y = cos(x)

Transformed Function

y = 1.3·cos(π/4·(x + 2.6)) + 0.35  {−1.8 ≤ x ≤ 3}

Transformations

Four transformations were applied to the parent cosine function to match the wave shape in the photo. The amplitude was increased to 1.3, stretching the curve vertically so the peaks and troughs are taller and deeper than the standard cosine wave. The period was changed to 8 by using π/4 as the coefficient inside the function — since 2π ÷ (π/4) = 8, meaning one full cycle takes 8 horizontal units. The curve was shifted 2.6 units to the left so the peak of the wave lines up with the correct part of the subject. Finally, a vertical shift of 0.35 units upward raises the midline of the wave slightly above the x-axis, reflecting the real-world position of the shape in the photo.

Domain & Range

The domain is restricted to {x ∈ ℝ | −1.8 ≤ x ≤ 3}, capturing the horizontal extent of roughly one full wave cycle visible in the photograph. The range is {y ∈ ℝ | −0.95 ≤ y ≤ 1.65}, which comes directly from the amplitude and vertical shift — the maximum is 1.3 + 0.35 = 1.65 at the peaks, and the minimum is −1.3 + 0.35 = −0.95 at the troughs. Both bounds align with what's physically visible in the image.

— Exponential Function

y = eˣ

Parent Function

y = eˣ

Transformed Function

f(x) = 0.3e^(−(x+1.4)) − 0.6  {−2 < x < 0.9}

Transformations

The parent exponential function y = eˣ went through four transformations to fit the curve in the photo. A vertical compression by a factor of 0.3 scales the output values down so the curve stays relatively flat rather than shooting up rapidly like a standard exponential. The negative exponent reflects the curve across the y-axis, flipping it so it decreases from left to right instead of increasing — giving it that decaying shape. A horizontal shift of 1.4 units to the left moves the curve into position over the subject in the photo. Finally, shifting down 0.6 units drops the horizontal asymptote from y = 0 to y = −0.6, keeping the curve just below the x-axis throughout the visible region.

Domain & Range

The domain is restricted to {x ∈ ℝ | −2 < x < 0.9}, corresponding to the horizontal width of the subject visible in the photograph. Within this interval the curve is smoothly decreasing and well-behaved. The range is {y ∈ ℝ | −0.58 < y < −0.17}, a narrow band of values just below the x-axis. This makes sense given the vertical compression and downward shift — the curve never strays far from its asymptote at y = −0.6, staying within a small range of negative output values across the entire domain.

Function 07

Eilyaz

— Logarithmic Function

y = log(x)

Parent Function

y = log(x)

Transformed Function

f(x) = 34·log₂(−100(x−17)) − 430  {x < 17}

Transformations

This one needed quite a few transformations to get the logarithmic curve to match the shape in the photo. A vertical stretch of 34 makes the curve much steeper than a standard logarithm — necessary because the shape in the image drops very sharply. The negative sign inside the logarithm reflects the curve across the y-axis so it decreases from left to right rather than increasing, giving it that falling shape. A horizontal compression by a factor of 100 squishes the curve so it changes very quickly over a small x range. The whole graph was then shifted 17 units to the right and 430 units downward, repositioning the steep section of the curve to match exactly where the shape appears in the photo.

Domain & Range

The domain is {x ∈ ℝ | x < 17}, which comes from the requirement that the argument inside the logarithm must be positive. Since the argument is −100(x − 17), it is only positive when x < 17, so the function only exists to the left of that point — which is also exactly where the visible shape in the photo is located. The range is all real numbers {y ∈ ℝ}, because as x approaches 17 from the left the output goes to negative infinity, and as x decreases the output increases without bound, with no restriction on vertical values within the domain.

Fountain — Quadratic Function

y = a(x−h)² + k

Parent Function

y = x²

Transformed Function

y = −0.04(x − 0)² + 1  {−10 ≤ x ≤ 10}

Transformations

A fountain is one of the most natural real-world examples of a parabola — the water jets arc upward and fall back down in a perfect curved path. Starting from the parent function y = x², two main transformations were applied. A vertical reflection through the negative leading coefficient flips the parabola downward to match the arching water jets, which go up then come back down. A vertical compression of 0.04 makes the curve very wide and shallow, matching the gentle arc of the spray rather than a sharp narrow peak. No horizontal shift was needed since the vertex sits naturally at the centre of the fountain, so h = 0. A vertical shift of 1 unit upward raises the vertex to (0, 1), aligning it with the maximum height the water reaches at the top of the arc.

Domain & Range

The domain is restricted to {x ∈ ℝ | −10 ≤ x ≤ 10}, representing the full horizontal spread of the fountain's water jets from one side to the other — 20 units total, centred at the origin. The range is {y ∈ ℝ | −3 ≤ y ≤ 1}, which covers the vertical extent of the parabola within the domain. The maximum of y = 1 is the vertex at the top of the arc, and the minimum at the edges of the domain comes from substituting x = ±10 into the equation: y = −0.04(100) + 1 = −3. This matches the physical reality of the water jets falling back down past the base level of the fountain.

MH

Staircase Railing — Linear Function

y = mx + b

Parent Function

y = x

Transformed Function

y = −0.7x + 4  {0 < x < 6}

Transformations

The staircase railing is one of the most straightforward real-world examples of a linear function — it's a perfectly straight line going diagonally across the photo. Starting from y = x, two transformations were made. The slope was changed to −0.7, which does two things at once: the negative sign reflects the line downward so it slopes from top-left to bottom-right, matching the direction the railing descends as you walk down the stairs; and the 0.7 compresses the steepness slightly so it isn't a full 45-degree angle. A vertical shift of 4 units upward was also applied by adding 4, moving the y-intercept up to 4 — positioning the top of the railing at a height of 4 units on the graph, which matches where it appears in the photo. The domain was restricted to between 0 and 6 to cover only the visible length of the railing.

Domain & Range

The domain is {x ∈ ℝ | 0 < x < 6}, representing the horizontal length of the staircase railing across the photo — approximately 6 metres from the base of the stairs to the top. The range is {y ∈ ℝ | −0.2 < y < 4}, representing the height of the railing above the ground across that distance. At x = 0 the railing starts at a height of 4 metres, and by x = 6 it has come down to about −0.2, which is effectively ground level. The slight negative lower bound is just a result of the linear equation extending slightly past ground level — in reality the railing ends right at the bottom of the stairs.
